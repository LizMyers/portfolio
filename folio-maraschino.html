ICON DESIGN + FRAMEWORK

Project Maraschino

ROLE: Lead Designer/Developer
DATE: Sep 2015 - Jan 2017
CLIENTS: Amazon, BlackBerry, Concur
TOOLS: AI, jQuery, JS, HTML, CSS
Challenge

Creating a family of glyphs that communicate well, are extensible, and easy to maintain presents an ongoing challenge for developers. Beyond the initial design of each icon, there's a larger problem of keeping track of the icon in it's many forms (outline, solid, vector, png).

What I Did

I was asked to create an icon library for BB10 developers. I delivered 250 glyphs that met the brand requirements and were rendered in 2 differenct color schemes (dark and light).Developers accessed the icons via a "cheet-sheet". But browsing became more time consuming as the library grew - I knew there had to be a better way.

Maraschino Framework

I built a data base, tagged each glyph with keywords and categories, built the jQuery framework, used masonary

Impact

BlackBerry, Amazon, Concur all hired me - Font Awesome changed... life is good :-)
